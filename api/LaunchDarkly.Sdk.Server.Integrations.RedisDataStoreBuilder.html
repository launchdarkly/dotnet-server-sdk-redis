Remarks

Obtain an instance of this class by calling DataStore(). After calling its methods to specify any desired custom settings, wrap it in a LaunchDarkly.Sdk.Server.Integrations.PersistentDataStoreBuilder by calling LaunchDarkly.Sdk.Server.Components.PersistentDataStore(LaunchDarkly.Sdk.Server.Interfaces.IPersistentDataStoreFactory), then pass the result into the SDK configuration with LaunchDarkly.Sdk.Server.ConfigurationBuilder.DataStore(LaunchDarkly.Sdk.Server.Interfaces.IDataStoreFactory). You do not need to call CreatePersistentDataStore(LdClientContext) yourself to build the actual data store; that will be done by the SDK.

Builder calls can be chained, for example: 

    var config = Configuration.Builder("sdk-key")
        .DataStore(
            Components.PersistentDataStore(
                Redis.DataStore()
                    .Uri(new Uri("redis://my-redis-host"))
                    .Database(1)
                )
                .CacheSeconds(15)
            )
        .Build();
