Remarks

This class is retained in version 1.2 of the library for backward compatibility. For the new preferred way to configure the Redis integration, see Redis. Updating to the latter now will make it easier to adopt version 6.0 of the LaunchDarkly .NET SDK, since an identical API is used there (except for the base namespace).
